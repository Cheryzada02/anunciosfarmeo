# Discord Anuncios Bot (discord.js v14)

## Features
- /anuncio solo para roles permitidos (ALLOWED_ROLE_IDS)
- Canal fijo por ID (ANNOUNCE_CHANNEL_ID) -> NO pregunta canal
- Modal para escribir anuncio: título + mensaje + imagen URL opcional
- Embed estético con icono del servidor, autor, timestamp
- Color del embed por config (EMBED_COLOR) estilo logo
- Keep-alive HTTP + auto-ping para Render
